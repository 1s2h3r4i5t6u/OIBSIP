Calculator Project
This is a simple calculator project created as part of the Oasis Internship at InfoByte.

Project Overview
The Calculator Project aims to develop a basic calculator application that can perform various mathematical operations. The calculator will have a user-friendly interface where users can input numbers and select the desired operation to perform.

Features
Addition: Perform addition of two or more numbers.
Subtraction: Perform subtraction of two numbers.
Multiplication: Perform multiplication of two or more numbers.
Division: Perform division of two numbers.
Clear: Reset the calculator and clear the display.
Error Handling: Display appropriate error messages for invalid inputs or operations.
