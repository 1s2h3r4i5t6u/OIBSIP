Oasis Infobyte To-Do List
Description
The Oasis Infobyte To-Do List is a simple web-based application that allows users to create and manage their to-do tasks. It provides an intuitive and user-friendly interface to help users stay organized and prioritize their tasks effectively.

Features
User Registration: Users can create an account to access the application.
Login: Registered users can log in to their accounts securely.
Create Task: Users can create new tasks by providing a title, description, and due date.
Task List: The application displays all the tasks in a list format, showing the task title, due date, and completion status.
Task Details: Users can view the details of a specific task, including the description and due date.
Update Task: Users can edit task details such as the title, description, and due date.
Mark as Complete: Users can mark a task as complete when they have finished it.
Delete Task: Users can delete a task if they no longer need it.
Filter Tasks: Users can filter tasks based on completion status or due date to easily find specific tasks.
Sort Tasks: Users can sort tasks by due date to prioritize their tasks effectively.
User Profile: Users can view and update their profile information.
